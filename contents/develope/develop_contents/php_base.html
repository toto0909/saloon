可変長引数

 
        
            何個引数に入っても対応できる(配列化して引数にいれる)
            function sum(...$numbers)
            {
              $total = 0;
              foreach ($numbers as $number) {
                $total += $number;
              }
              return $total;
            }

複数の返り値

 
        
           返り値を配列にして複数の情報を返す
           function getStats(...$numbers){
                $total = 0;
                foreach ($numbers as $number) {
                    $total += $number;
                }
                return [$total, $total / count($numbers)];
                }

                配列形式で受け取る
                list($sum, $average) = getStats(1, 3, 5);　もしくは
                [$sum, $average] = getStats(1, 3, 5)
